📄 CSV Format Guide

Players CSV Format

Required columns: name, position

Optional columns: jersey_number, age, nationality

name position jersey_number age nationality
John Smith Forward 9 25 England
Mike Johnson Midfielder 8 28 Spain

Matches CSV Format

Required columns: match_date, opponent, venue

Optional columns: team_goals, opponent_goals, possession, shots, shots_on_target, corners, fouls

match_date opponent venue team_goals opponent_goals possession
2024-01-15 Team A Home 3 1 58.5
2024-01-22 Team B Away 2 2 45.0

💡 Tips for Successful Upload

  • Date Format: Use YYYY-MM-DD format for dates (e.g., 2024-01-15)
  • Venue Values: Use "Home" or "Away" for the venue column
  • Position Values: Use standard positions like "Forward", "Midfielder", "Defender", "Goalkeeper"
  • Encoding: Save your CSV file with UTF-8 encoding for best compatibility
  • Headers: Make sure the first row contains column headers exactly as shown above
